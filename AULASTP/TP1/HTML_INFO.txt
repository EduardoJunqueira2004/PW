O HTML é uma linguagem de marcas: ele usa tags para definir diferentes partes do conteúdo.

Em html temos atributos e elementos diz-me o que é um atributo e um elemento?
A: Atributo é uma propriedade que é aplicada a um elemento. Elemento é uma tag que é usada para marcar o conteúdo.
Exemplos de elementos: <p> <h1> <a> <img> <ul> <ol> <li> <table> <tr> <td> <th> <form> <input> <button> <select> <option> <label> <span> <div> <br> <hr>
Exemplos de atributos: href, src, alt, width, height, type, value, name, id, class, style, action, method, for, checked, selected, disabled, readonly, placeholder, required, min, max, step, autocomplete, autofocus, multiple, pattern, novalidate, form, target, rel, title, lang, dir, tabindex, accesskey, hidden, draggable, contenteditable, spellcheck, role, aria-*, data-*
